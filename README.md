# Internet-in-a-Box (IIAB)

Welcome to the Git repository of the Internet-in-a-Box (IIAB) project. This is a community-based project developed and supported by volunteers from around the world. The Internet-in-a-Box(IIAB) is small, inexpensive device which provides essential Internet resources without any Internet connection.
