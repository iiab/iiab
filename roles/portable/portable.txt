portable.yml used to create a containter filesystem to hold the contents of /library gaining the abiltiy to export the entire data structure as one file living somewhere else on the filesystem.
library_loc: path to image, defaults to '{{ iiab_base }}'
library_img: name of the file holding the underling filesystem.
library_size: passed directly to dd use the shorthand such as 115g 10G 500M to add size to the /library 'partition' 
You don' have to use all that is available starting with 3G works with small install with half still available for testing in a VM 10GB.
library_base_url: web location of 100MB pre-populated image file with only admin-console pre-installed.
library_base_path: local location of pre-popuplated image file to expand.
library_url: web location of completed pre-populated image file to deploy.
